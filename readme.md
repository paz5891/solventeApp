# CAMBIO EN COMANDO DE EJECUTAR EL SERVER REST API

- para usarlo localmente debe de ejecutar el siguiente comando

    ```
    npm run dev
    ```
    **nota**: ese serviara para ejecutarlo con modulo nodemon

    para ejecutar en entorno de produccion o en una pagina debe de ejecutar el siguiente comando

    ```
    npm start
    ```
    ## NOTA
    -   temporalmente de usara el documento **.env** para desarrollar cuando se implemente en produccion se estableceran como variables de entorno las cadenas que estan dentro de ese documento
    
    - en proceso que sucede cuando se ejecuta el comando **npm start** se crea una carpeta llamada **build** lo que hace es hacer un conversor de de javascript para que lo pueda leer cualquier navegador
